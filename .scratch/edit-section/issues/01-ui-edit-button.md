# Add Edit Section Button to Timetable View

## Requirements
- Add an "Edit Section" button (or pencil icon) next to the section names on the main Timetable view.
- This applies to both the `webapp` and `mobile` projects.
- When clicked, it should:
  1. Store the currently viewed roll number (e.g. in `temp_linking_roll_no`).
  2. Navigate the user to the existing Section Selection screen (`/select/sections`).
- This initiates the Edit Section flow.
