# KIIT Time Backend

FastAPI backend for KIIT Time — a data pipeline and API serving
timetable data to the KIIT Time Android app and web PWA, with an admin
intake/approval workflow for timetable uploads.

Stack: Python 3.14, FastAPI, SQLAlchemy 2.0 + Alembic, Pydantic v2, uv.

See the main repository for full project context:
https://github.com/arthishaxom/kiittime
